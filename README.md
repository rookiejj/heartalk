# 두근카드

커플도, 친구도. 카드를 뽑고 대화를 나눠요.

## 소개

카드 한 장으로 대화가 시작돼요. 말이 많아질수록 마음도 깊어져요.

### 카드 종류

- **친구 카드** — 워밍업 / 밸런스 게임 / TMI (각 30장)
- **커플 카드** — 탐구 / 사랑 / 미래 (각 30장)
- **19금 카드** — Yes or No / 깊은 대화 / 미션 (총 91장 셔플)

## 사용법

카드를 탭하면 질문이 공개되고, 다시 탭하면 다음 카드로 넘어가요.

## 기술 스택

- HTML / CSS / JavaScript (Vanilla)
- Vercel (배포)

## 배포

```bash
git clone https://github.com/rookiejj/heartalk.git
cd heartalk
# index.html을 브라우저로 바로 열거나 Vercel로 배포
```
